# 🚀 Hello LLM

Простой CLI-скрипт для отправки запросов в LLM через GigaChat API.

---

## 📦 Подготовка

Перед запуском необходимо задать токен через системную переменную:

```bash
export GIGACHAT_AUTH_KEY="some_key"
```

---

## ▶️ Запуск

Запустите скрипт:

```bash
python3 hello_llm.py
```

---

## 💬 Использование

После запуска введите ваш запрос в терминале:

```text
Введите запрос:
```

Скрипт отправит запрос в модель и выведет ответ в консоль.

---

## 🛠 Требования

* Python 3.10+
* Доступ к GigaChat API
* Установленные зависимости проекта
