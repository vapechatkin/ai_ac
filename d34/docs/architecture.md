# Архитектура d34

```text
цель пользователя
       │
       ▼
 assistant.py → FileAgent / Claude tool-use loop
                         │
                         │ MCP over stdio
                         ▼
                    mcp_server.py
       ┌────────────┬──────────┬───────────┬──────────┐
       │ list/search│ read     │ write     │ check/diff│
       └────────────┴──────────┴───────────┴──────────┘
                         │
                         ▼
                 выбранный project root
```

Claude видит только относительные пути и результаты MCP-инструментов. Сервер
канонизирует каждый путь и отклоняет абсолютные пути, `..` за пределы корня и
служебные каталоги. Запись выполняется через временный файл и атомарный `replace`.

Сервер сохраняет исходное содержимое изменённых файлов в памяти процесса, поэтому
`diff_changes` строит единый diff всей сессии. Повторная запись идентичного текста
возвращает `changed: false`, что делает повторение задачи безопасным.
